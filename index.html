Snake Game

Score: 0

How to Play

Use the arrow keys or swipe on mobile to move the snake:

  • Left: Arrow Left / Swipe Left
  • Up: Arrow Up / Swipe Up
  • Right: Arrow Right / Swipe Right
  • Down: Arrow Down / Swipe Down

The objective is to eat the yellow food blocks and grow the snake. Avoid hitting the walls or the snake's own body.
